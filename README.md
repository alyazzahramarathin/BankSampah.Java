## Deskripsi
Aplikasi Bank Sampah berbasis Java dan MySQL yang dibuat untuk memenuhi Tugas Akhir Mata Kuliah Pemrograman II.

## Fitur
- Login
- CRUD Data Nasabah
- CRUD Data Sampah
- CRUD Data Transaksi
- Cetak Laporan

## Teknologi
- Java
- NetBeans IDE
- MySQL
- JDBC
- MVC

## Cara Menjalankan
1. Import database db_banksampah.sql
2. Buka project di NetBeans
3. Jalankan aplikasi
